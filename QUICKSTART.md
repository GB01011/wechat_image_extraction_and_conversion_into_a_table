# 快速启动指南 (5分钟内搞定)

## 📝 完整步骤清单

### 第1分钟：激活虚拟环境

```bash
# 进入项目目录
cd d:\acceptance_file\one\WeChat_Excel_Automator

# 激活虚拟环境
interpreter\Scripts\activate

# ✓ 命令行前会出现 (interpreter) 标志
```

### 第2分钟：安装依赖（如未安装）

```bash
# 仅第一次需要做
pip install -r requirements.txt
```

### 第3-4分钟：修改配置文件

**用任何文本编辑器打开** `config/settings.yaml`：

```yaml
# 改这里 ↓（改成你的微信好友或群名）
wechat:
  listen_list:
    - "你的好友名"   # 例如："小王"
    - "你的群名"     # 例如："采购部"

# 改这里 ↓（填入真实的 API Key）
llm:
  api_key: "sk-...从阿里云或OpenAI申请..."
  base_url: "https://dashscope.aliyuncs.com/compatible-mode/v1"  # 阿里云
  model_name: "qwen-vl-plus"
```

### 第5分钟：启动程序

```bash
# 确保虚拟环境已激活
python main.py
```

## 🚨 常见错误快速排查

| 错误 | 原因 | 解决 |
|------|------|------|
| `ModuleNotFoundError: No module named 'wxauto'` | 依赖未安装 | `pip install -r requirements.txt` |
| `"无法监听【xxx】"` | 好友名错误 | 检查拼写（大小写、符号都要一致） |
| `"API 调用失败"` | API Key 错误或示例值 | 填入真实的 API Key |
| `"微信初始化失败"` | 微信未运行或未登录 | 确保微信已打开且完成登录 |
