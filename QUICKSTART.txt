## ❓ 常见问题

### Q: 还需要安装 wxauto 吗？
**A**：不需要！已完全移除。本地模式无需 wxauto。

### Q: 支持批量处理吗？
**A**：支持！放多张图片到 `data/temp_images` 会自动批量处理。

### Q: API Key 用错可以重新配置吗？
**A**：可以。直接修改 `config/settings.yaml` 中的 API Key，重新运行 `python main.py`。

### Q: 如何切换到微信直连模式？
**A**：修改 `config/settings.yaml`：
```yaml
wechat:
  mode: "wechat"  # 改这里
  listen_list:
    - "你的好友名"
    - "你的群名"
```
然后安装：`pip install wxauto`
